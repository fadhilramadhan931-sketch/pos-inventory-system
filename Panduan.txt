## 📋 Menu Utama

### 1. Transaksi Baru
- Pilih produk
- Input jumlah
- Diskon otomatis dihitung
- Bayar dan cetak struk

### 2. Kelola Produk
- Lihat semua produk
- Tambah produk baru

### 3. Lihat Stok
- Inventory tracking
- Alert stok rendah

### 4. Laporan
- Laporan harian
- Top 10 produk terlaris

## 💰 Diskon Otomatis

- Total > Rp 500.000 → 15% diskon
- Total > Rp 200.000 → 10% diskon  
- Total > Rp 100.000 → 5% diskon
- Total ≤ Rp 100.000 → Tanpa diskon

## 🧮 Calculation

- PPN otomatis 11%
- Kembalian otomatis dihitung
- Receipt dicetak otomatis

## 💾 Database

- File: `pos_system.db` (SQLite)
- Auto-created pada run pertama
- Simpan otomatis setiap transaksi

## 🎨 Fitur

✅ Stock tracking  
✅ Transaction history  
✅ Reports    
✅ Input validation  

## 🔧 Troubleshooting

### Error "Module not found"
- Pastikan Python 3 terinstall
- Check path file

### Database error
- Hapus file `pos_system.db`
- Run ulang program

### Terminal warna tidak muncul (Windows)
- Normal, warna hanya jalan di Linux/Mac
- Fungsionalitas tetap berjalan normal

## 📝 Notes

- Data otomatis tersimpan di database
- Bisa dijalankan berkali-kali
- Data tidak hilang
- Cocok untuk toko kecil/menengah
